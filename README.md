#NIDS Capstone Project

## Setup
1. Make sure virtual environment is activated: 'source nids_env/bin/activate'
2. Install dependencies: 'pip install -r requirements.txt' or just 'requirements' in my case

## Project Structure
- '/data': Raw and processed datasets
- 'src/': Python source code


- 'models/': Saved trained models
- 'results/': Evaluation results and graphs
- 'notebooks/': Jupyter notebooks for exploration purposes
